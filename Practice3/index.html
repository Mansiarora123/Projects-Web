Understanding CSS: The Basics

By Mansi Arora | Published: June 1, 2025

Introduction to CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML. CSS controls the layout of multiple web pages all at once.

Why Use CSS?

Some key benefits of CSS include:

  • Separation of content and design
  • Improved website maintenance and consistency
  • Flexible layouts and responsive design

Basic CSS Syntax

CSS syntax consists of selectors and declarations:

  1. Selector: The HTML element you want to style
  2. Declaration: The property and value to apply

Example:

h1 {
  color: blue;
  font-size: 2em;
}

CSS can be included inline, internally within a style tag, or externally via CSS files. External CSS is the preferred method for scalability.
